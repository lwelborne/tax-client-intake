Overview

The Tax Client Intake Form is a full-stack web application designed to simplify how tax professionals collect and manage client information. Built with React, Node.js, and MongoDB, the app provides an easy-to-use interface for entering client data, editing records, and viewing submissions.

This project demonstrates end-to-end full-stack development skills, from designing a responsive user interface to building secure backend APIs and integrating a cloud database. By digitizing the intake process, it reduces manual paperwork, increases accuracy, and ensures client data is securely stored and easy to access.

Tech Stack

  • Languages & Formats: JavaScript, HTML, CSS
  • Frameworks & Libraries: React, Node.js, Express
  • Database & Hosting: MongoDB Atlas, Vercel (frontend), Render (backend)
  • Version Control & Tools: Git, GitHub, Postman
